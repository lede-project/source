# openwrt-dogcom
[dogcom](https://github.com/mchome/dogcom.git) is [drcom-generic](https://github.com/drcoms/drcom-generic) implementation in C.

### To build:
```bash
$ make package/openwrt-dogcom/compile
```

### Notice:
This package do not include a init file.
